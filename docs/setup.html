Setup is easy

See the elements with dotted borders? Hover your cursor over them to see the variables that set them accordingly.

More variables

The download links on the left are generated using site.repository_url

site.title is shown as the title in tabs. page.title may be used to overide site.title

page.excerpt may also be used to add description of page.

Page Navigation

To create a navigation bar as the one at the top of the page, you need to place an mapping of titles to relative urls under the key items in the file _data/menu.yml

For example,


items:
  "introduction": "/"
  "setup": "/setup/"

Docs

You can simply place any markdown files in the docs/ directory for your documentation. They will be automatically converted to HTML.

If need be, you may use HTML files instead.

A Sample file would be: (docs/sample.md)


---
layout: default
permalink: /endpoint
---

# markdown

I shall write this in Markdown only.

Done with Setup

You may remove docs/setup.html and script/ as they are not required for deployment.
